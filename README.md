# Sales Dashboard 🚀

A modern sales dashboard with AI insights and database support. By default, it reads data from a local CSV file, but it is fully ready to connect to Supabase whenever you are ready.

---

## ⚡ Quick Connect: Supabase (Non-Coder Guide)

Follow these simple steps to move your data from a spreadsheet to a real database:

### 1. Create your Project
Go to [Supabase](https://supabase.com/) and create a new project. It takes about 2 minutes.

### 2. Create the Table
In your Supabase project, go to the **SQL Editor** (the `>_` icon on the left) and paste/run this code:

```sql
-- Create the sales_data table
CREATE TABLE sales_data (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  date DATE NOT NULL,
  product TEXT NOT NULL,
  channel TEXT NOT NULL,
  orders INTEGER DEFAULT 0,
  revenue DECIMAL(12,2) DEFAULT 0,
  cost DECIMAL(12,2) DEFAULT 0,
  visitors INTEGER DEFAULT 0,
  customers INTEGER DEFAULT 0
);

-- Allow the dashboard to see the data
ALTER TABLE sales_data ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Allow public read" ON sales_data FOR SELECT TO anon USING (true);
```

### 3. Import your Data
1. Go to the **Table Editor** in Supabase and select your new `sales_data` table.
2. Click **Insert** -> **Import data from CSV**.
3. Upload your `sales_data.csv` file.

### 4. Connect the Dashboard
Open the [`.env`](file:///c:/Users/YZ/Downloads/vc-dashboard/.env) file in this folder and update these lines:

```env
# Switch from 'csv' to 'supabase'
VITE_DATA_SOURCE=supabase

# Your project details
VITE_SUPABASE_URL=your_project_url
VITE_SUPABASE_ANON_KEY=your_anon_key
```

**Where to find these in Supabase?**
1. Click the **Settings** (Gear icon) at the bottom of the left sidebar.
2. Under "Project Settings", click on **API**.
3. Under **"Project URL"**, copy the URL (it looks like `https://xxx.supabase.co`). This is your **VITE_SUPABASE_URL**.
4. Under **"Project API keys"**, look for the key labeled **`anon`** and **`public`**. Copy it. This is your **VITE_SUPABASE_ANON_KEY**.

### 5. Restart
Stop your terminal (**Ctrl + C**) and run:
`npm run dev`

**That's it! Your dashboard is now running on a professional database.** 🎊
